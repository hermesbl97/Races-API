# Races-API
Se trata de una aplicación que permite mostrar, registrar, modificar y eliminar carreras, atletas e inscripciones. Una inscripción se encuentra vbinculada tanto a una carrera como a un atleta. 

La aplicación está compuesta por un backend en el que se encuentra una API-REST y un frontend que externaliza la información. La API se encarga de gestionar las peticiones y gestiona los datos almacenados en una base de datos Sqlite3. 

El lenguaje empleado para programar esta aplicación es js. La aplicación permite registrar fotos de los atletas registrados y en su defecto almacena una imagen por defecto en la base de datos. Se han implementado tanto validaciones como notificaciones de éxito o de error para que el usuario pueda conocer si su acción se ha realizado de manera correcta.
